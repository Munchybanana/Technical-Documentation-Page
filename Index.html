Introduction

Hello, you may be wondering "What is HTML?" well HTML, or also known as HyperText Markup Language, is the standard language for documents to be displayed in a web browser. HTML is also assisted greatly by another language, Cascading Style Sheets or CSS for short, and scripting languages such as JavaScript.

HTML was first written by Tim Bernes Lee, in 1993! However it would not be published until 1995.

What you should already know

Most of the content in this guide is presented to you, assuming you know the following.

  • General understanding of the internet, and the World Wide Web (WWW)
  • Basic understanding of how to use a computer
  • An internet connection
HTML and CSS

As mentioned earlier, HTML and CSS are very important, and are used together all the time. CSS simply will help you (the developer) visually design your site, and give you the tools necessary for it. This is why its recommended you learn these two languages first.

For example, CSS allows you to make adjustmants to your text, for example if you don't want your text on the left, then you can simply use text-align: center; which will allow you to center that text

Headings

Headings are important, since they tell the browser the structure of your website. <h1> Are often used for main headings. <h2> Elements are generally used for subheadings. There are also <h3>,<h4>,<h5>, and <h6> which are different levels of subheadings. Example:

<h1>Heading Level 1</h1> <h2>Heading Level 2</h2> <h3>Heading Level 3</h3> <h4>Heading Level 4</h4> <h5>Heading Level 5</h5> <h6>Heading Level 6</h6>

The output:

Heading Level 1

Heading Level 2

Heading Level 3

Heading Level 4

Heading Level 5
Heading Level 6
Paragraph

Paragraph elements are preferred for paragraph text on websites. Paragraph elements are created like this:

<p> I'm a p tag! </p>

The output:

I'm a p tag!

Line Break

The line break element produces a line break in the text. This can be useful for when you are writing something, such as a poem. For example:

<p> I <br> have <br> line <br> breaks. </p>

The output:

I
have
line
breaks.

Links

Links are used with Anchor elements. Links can be added through a number of ways.

  • With an Anchor, you can give it a link, then display that link with Anchor text Like this!
  • Anchor elements can also be used to create internal links to jump to different sections within a webpage.
  • You can also use an Anchor element on an image, the image will then act like a link as well.

Here are a few examples of an Anchor element being used:

<p> Hello, <a href="#"> I'm Anchor text within a paragraph element! </a> </p> <a href="https://www.google.com" target="_blank"> I open a new tab to google! </a>

The output:

Hello, I'm Anchor text within a paragraph element!

I open a new tab to google!
Comments

Comments are a great way to leave messages for other developers within your code without affecting the output to the user. Commenting is also a good to make sections of code inactive, without deleting it.

<!-- This is a comment --> <!-- <h1> Hello World </h1> -->
Lists

There are two lists that you can create, you can create Bulleted Unordered List, or Ordered List. Lists can be useful, since they can let you organize any info you want into a list, helping the user read through content, or navigate a site.

Here is an example of a Bulleted Unordered List:

<ul> <li> List 1 </li> <li> List 2 </li> <li> List 3 </li> </ul>

The outcome:

  • List 1
  • List 2
  • List 3

Here is an example of a Ordered List:

<ol> <li> List 1 </li> <li> List 2 </li> <li> List 3 </li> </ol>

The outcome:

  1. List 1
  2. List 2
  3. List 3
Font Style

There are a few different font styles that you can do.

  • Bold:

To make a word or sentence bold you use <strong> </strong>

Here is an example:

<strong> This sentence is bold. </strong>

Output:

This sentence is bold.
  • Italic:

To make a word or sentence italic, you use

<i> </i>

Here is an example:

<i> This sentence is italic. </i>

Output:

This sentence is italic.
  • Underline:

To make a word or sentence underlined, you use

<u> </u>

Here is an example:

<u> This sentence is underlined </u>

Output:

This sentence is underlined
  • Super Script:

To make a word or sentence with super script, you use

<sup> </sup>

Here is an example:

<p> a<sup>2</sup>+ b<sup>2</sup>= c<sup>2</sup> </p>

Output:

a2 + b2 = c2

  • Subscript:

To make a word or sentence with subscript, you use

<sub> </sub>

Here is an example:

<p> My favorite drink is H<sub>2</sub>O. </p>

Output:

My favorite drink is H2O.
